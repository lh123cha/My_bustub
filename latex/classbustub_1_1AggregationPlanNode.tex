\doxysubsection{Detailed Description}
\mbox{\hyperlink{classbustub_1_1AggregationPlanNode}{Aggregation\+Plan\+Node}} represents the various S\+QL aggregation functions. For example, C\+O\+U\+N\+T(), S\+U\+M(), M\+I\+N() and M\+A\+X().

N\+O\+TE\+: To simplify this project, \mbox{\hyperlink{classbustub_1_1AggregationPlanNode}{Aggregation\+Plan\+Node}} must always have exactly one child.
