\doxysubsection{Detailed Description}
Debug logging functions for EE. Unlike the performance counters, these are just fprintf() turned on/off by L\+O\+G\+\_\+\+L\+E\+V\+EL compile option. The main concern here is not to add any overhead on runtime performance when the logging is turned off. Use L\+O\+G\+\_\+\+X\+X\+X\+\_\+\+E\+N\+A\+B\+L\+ED macros defined here to eliminate all instructions in the final binary. \begin{DoxyAuthor}{Author}

\mbox{\hyperlink{lock__manager__test_8cpp}{lock\+\_\+manager\+\_\+test.\+cpp}}

This file contains basic tests for the functionality of all nine executors required for Fall 2021 Project 3\+: Query Execution. In particular, the tests in this file include\+:


\begin{DoxyItemize}
\item Sequential Scan
\item Insert (Raw)
\item Insert (Select)
\item Update
\item Delete
\item Nested Loop Join
\item Hash Join
\item Aggregation
\item Limit
\item Distinct
\end{DoxyItemize}

Each of the tests demonstrates how to construct a query plan for a particular executors. Students should be able to learn from and extend these example usages to write their own tests for the correct functionality of their executors.

Each of the tests in this file uses the {\ttfamily \mbox{\hyperlink{classbustub_1_1ExecutorTest}{Executor\+Test}}} unit test fixture. This class is defined in the header\+:

{\ttfamily \mbox{\hyperlink{executor__test__util_8h}{test/execution/executor\+\_\+test\+\_\+util.\+h}}}

This text fixture takes care of many of the steps required to set up the system for execution engine tests. For example, it initializes key D\+B\+MS components, such as the disk manager, the buffer pool manager, and the catalog, among others. Furthermore, this text fixture also populates the test tables used by all unit tests. This is accomplished with the help of the {\ttfamily \mbox{\hyperlink{classbustub_1_1TableGenerator}{Table\+Generator}}} class via a call to {\ttfamily Generate\+Test\+Tables()}.

See the definition of {\ttfamily \mbox{\hyperlink{classbustub_1_1TableGenerator_a38027e024ff7fc2e74a656ca72d9bc5f}{Table\+Generator\+::\+Generate\+Test\+Tables()}}} for the schema of each of the tables used in the tests below. The definition of this function is in {\ttfamily \mbox{\hyperlink{table__generator_8cpp}{src/catalog/table\+\_\+generator.\+cpp}}}.
