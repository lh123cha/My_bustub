\doxysubsection{Detailed Description}
class \mbox{\hyperlink{classbustub_1_1IndexMetadata}{Index\+Metadata}} -\/ Holds metadata of an index object.

The metadata object maintains the tuple schema and key attribute of an index, since the external callers does not know the actual structure of the index key, so it is the index\textquotesingle{}s responsibility to maintain such a mapping relation and does the conversion between tuple key and index key
