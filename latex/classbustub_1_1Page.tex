\doxysubsection{Detailed Description}
\mbox{\hyperlink{classbustub_1_1Page}{Page}} is the basic unit of storage within the database system. \mbox{\hyperlink{classbustub_1_1Page}{Page}} provides a wrapper for actual data pages being held in main memory. \mbox{\hyperlink{classbustub_1_1Page}{Page}} also contains book-\/keeping information that is used by the buffer pool manager, e.\+g. pin count, dirty flag, page id, etc.
