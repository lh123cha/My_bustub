Detailed Description

Slotted page format:

| HEADER | ... FREE SPACE ... | ... INSERTED TUPLES ... |

^ free space pointer

Header format (size in bytes):

| PageId (4)| LSN (4)| PrevPageId (4)| NextPageId (4)| FreeSpacePointer(4) |

| TupleCount (4) | Tuple_1 offset (4) | Tuple_1 size (4) | ... |
